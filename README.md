## Квартиры в России
Квартиры в России - веб приложение для поиска недвижимости в удобном районе.

## Стек
**Frontent**: JavaScript, React, Openlayers, HTML, CSS  
  
**Backend**: Java, Spring Boot, Spring MVC, Spring JPA  
  
**SQL**: h2 database  
  
## Запуск
#### Сборка:  
- mvn clean package  
- mvn spring-boot:run
